    Web Share Target API (PWA only)
    

    Allows your web app to receive shared data — e.g. text or images — from other apps.
(Works only when installed as a Progressive Web App.)

Manifest example:
    
{
  "share_target": {
    "action": "/share",
    "method": "POST",
    "enctype": "multipart/form-data",
    "params": {
      "title": "title",
      "text": "text",
      "url": "url",
      "files": [{ "name": "file", "accept": ["image/*"] }]
    }
  }
}
